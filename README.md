## Getting started

1. Download the repo

2. Install depencies

```bash
npm i
#or
yarn install
```
3. Create .env file and update .env file with below details

```bash
DB_HOST=
DB_PORT=
DB=
DB_USER=
DB_PASS=
```

4. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
